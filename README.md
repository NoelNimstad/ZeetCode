# ZeetCode

ZeetCode is a repository of Zig (v0.16.0) implementations of [LeetCode®](https://leetcode.com/) problems.
Tests cases which can be used along side with a function outline for the solution are found in the [problems](./problems/) directory.
Example solutions to the respective problems can be found in the [solutions](./solutions/) directory.
