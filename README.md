# ZeegCode

ZeegCode is a repository of Zig (v0.16.0) implementations of [LeetCode®](https://leetcode.com/) problems.
Tests which can be used a long with a function outline are found in the [problems](./problems/) directory.
Example solutions to the respective problems can be found in the [solutions](./solutions/) directory.
